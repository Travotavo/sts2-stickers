# Sticker the Spire
A Sticker Mod for Slay the Spire 2!

https://github.com/user-attachments/assets/e0f25fb7-9ede-4acd-aac2-93c469d13fa0

## Features
- 13 Custom Stickers to place on Cards.
- Sticker placements saved across quits.

## Planned Changes
- Easy support for mods to introduce more stickers.
- Sticker Order in place of current alphabetic sort.
- Avoid crashes for depreciated sticker names.
- In-game instructions for placing stickers.
- Rotating Stickers
- UI/UX design.
- Stickers for Act 2 and 3 Bosses.
- Rendering Ceremonial Beast, Kin, and Ironclad stickers.
- Controller Support
